Interdist Analytics System

Web app thu thập, xử lý và trực quan hóa dữ liệu bán hàng P&G cho hai kênh CRV (GO!/Big C) và STMB (Lan Chi). Tài liệu này hướng dẫn cách dùng từng màn hình của hệ thống.

Môi trường
Web App (chạy trên trình duyệt)
Stack
React · Vite · TypeScript · Firebase
Charts
Recharts (tương tác)
Phiên bản
Bản nghiệm thu
01

Giới thiệu

App đọc trực tiếp file báo cáo Excel/CSV của nhà phân phối, chuẩn hóa và dựng dashboard số liệu cho hai kênh. Toàn bộ xử lý chạy trên trình duyệt (client-side); xác thực và cấu hình lưu trên Firebase.

🎯

Mục đích

Biến file Excel raw thành dashboard doanh số & target real-time, giảm thao tác thủ công, chuẩn hóa cách theo dõi hai kênh CRV / STMB.

🧩

Cấu trúc 2 kênh

CRV BA Long Term — hệ thống GO!/Big C (~40 store, 6 vùng). STMB — Lan Chi Mart (~10 store). Hai kênh xem riêng hoặc gộp chung.

02

Đăng nhập & phân quyền

Đăng nhập bằng Email/Mật khẩu. Mỗi tài khoản có một trong ba cấp quyền, quyết định việc có thấy Cổng Dữ Liệu và quản lý người dùng hay không.

dev

Developer — Admin hệ thống

Toàn quyền: cấp/hạ quyền cho user khác, can thiệp cấu hình. Thấy đầy đủ Cổng Dữ Liệu & quản lý user.

admin

Quản trị viên

Thêm/xóa tài khoản cấp user, chỉnh cấu hình Target. Không đổi được vai trò người ở cấp admin/dev. Có Cổng Dữ Liệu.

user

Người dùng tiêu chuẩn

Chỉ xem báo cáo. KHÔNG thấy nút Cổng Dữ Liệu, không sửa cấu hình, không quản lý người dùng.

04

Màn hình Overall & các thẻ KPI

Sáu thẻ KPI chính. Bấm vào thẻ CRV / STMB để lọc nhanh theo kênh tương ứng.

Actual SO MTD
Doanh số lũy kế (tỷ VNĐ) kèm target tổng.
%ACH Full Month
% đạt target cả tháng. Ngưỡng on-track ≥ 40%.
CRV Channel
Doanh số riêng kênh CRV (GO!/Big C).
STMB Channel
Doanh số riêng kênh STMB (Lan Chi).
Stores Active
Số store đang hoạt động / tổng số store.
Daily Orders
Doanh số trung bình mỗi ngày (triệu VNĐ).
📉

Biểu đồ & bảng dưới KPI

  • Trend chart Target vs Actual theo ngày (tương tác — hover xem chi tiết)
  • Bảng theo Vùng kèm % tiến trình cho cả 2 kênh
  • Đổi bộ lọc Kênh / Vùng / Kỳ hạn → số liệu & chart cập nhật ngay

05

Detail reports

Báo cáo chi tiết, gồm các bảng đối chiếu theo nhiều chiều. Đây là nơi soi sâu khi cần kiểm tra số.

🏷

Theo tuần

  • Doanh số theo Ngành hàng / tuần
  • Doanh số theo Supervisor / tuần
📅

Cả tháng

  • Ngành hàng: Target Full · Actual MTD · % Hoàn thành · Gap vs Timegone
  • Vùng: cùng cấu trúc cột như trên
∑

Lũy kế MTD

  • Ngành hàng: Target MTD · Actual MTD · % Hoàn thành MTD
🕑

Ca làm (Shifts)

Riêng Detail reports có thêm kỳ hạn "Shifts" để xem theo ca làm việc của store.

📌
Gap vs Timegone: so sánh tiến độ thực tế với mốc thời gian đã trôi qua trong tháng — đây là chuẩn on/off track. Khi đối chiếu, để ý cột này khớp với cách tính bên file gốc.
06

Cổng Dữ Liệu (nạp file)

Chỉ dev / admin thấy được. Đây là nơi nạp file báo cáo mới vào hệ thống.

Bấm Cổng Dữ Liệu ở sidebar .
Kéo thả file vào vùng nhận, hoặc bấm chọn file. Hỗ trợ .xlsx .xls .xlsb .csv — nạp được nhiều file cùng lúc.
App tự chuẩn hóa tên cửa hàng, ngành hàng, mã nhân sự và tính các chỉ số (Revenue, Target, Shifts).
Sau khi nạp, quay lại Overall đối chiếu số mới có khớp file vừa import không.
⚠️
App tự fallback Mã vùng nếu file thiếu, và tự tạo Profile Target cho store mới/lạ. Nhưng nếu file đầu vào sai/thiếu giá hoặc thiếu target ngày, số trên dashboard sẽ lệch — luôn kiểm tra chất lượng file config trước.
07

Configure (cấu hình)

Quản lý dữ liệu nền của hệ thống. Bốn nhóm cấu hình, lưu vĩnh viễn trên Firebase.

💲

Bảng Giá Sản Phẩm

Giá tham chiếu theo SKU + ngành hàng. Cần đủ giá cho mọi SKU để không bị thiếu doanh số.

👤

Phân Công Supervisor

Gán Supervisor & Vùng cho từng mã cửa hàng.

🎯

Quản Lý Target Store

Target gốc từ Excel vs target áp dụng thực tế, theo từng dự án & store.

🔁

Nhập & Xuất Excel

Tải file Excel cấu hình (.xlsx) để cập nhật hàng loạt thay vì nhập tay.

08

Export & Telegram

📄

Export Report (PDF) (Chưa bắt đầu)

Xuất báo cáo điểm bán ra PDF sạch để gửi khách hàng P&G. Có nút "Copy nội dung" copy bản text vào clipboard.

📗

Export Excel

Chọn kênh + kỳ hạn, xuất file .xlsx (dạng dọc) để theo dõi offline.

✈️

Telegram Broadcast (Chưa bắt đầu)

Soạn & gửi nhanh tin nhắn thông báo cho đội Sales qua Telegram.
